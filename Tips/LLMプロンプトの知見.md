# 「～しないでください」だと全然従ってくれないが、「～することは禁止されています」だと高確率で従ってくれる
```
LLM向けのプロンプトで「～しないでください」のお願いベースだと全然指示従ってくれないが、「～することは禁止されています」だとかなり従ってくれる。
英文上のニュアンスに影響受けてそうな気配がある。
```
